# 一、最终验收清单

请建立或更新：

```text
docs/animation/CoreCat_动画系统实现验收报告.md
```

报告必须包含以下内容：

## 1. 分层骨骼验收

检查：

1. shadow 是否独立节点。
2. tail 是否独立节点。
3. body_base 是否独立节点。
4. arm_left 是否独立节点。
5. arm_right_wrench 是否独立节点。
6. head_base 是否独立节点。
7. ears_left / ears_right 是否独立节点。
8. goggles 是否独立节点。
9. eyes 是否独立节点。
10. pouch 是否独立节点。
11. vfx_anchor 是否独立节点。
12. 是否仍存在整张死图实现风险。

## 2. 核心动画状态验收

逐项检查：

1. Idle
2. Hover
3. Click
4. TemperatureCheck
5. MemoryCrowded
6. Repairing
7. DataSorting
8. Sleep / LowPower
9. Celebrate

每项写明：

1. 是否已实现骨骼 pose。
2. 是否已实现专属 VFX。
3. 是否可在 Debug Panel 手动触发。
4. 是否可自动回落。
5. 是否存在待正式美术替换项。

## 3. 补充状态验收

逐项检查：

1. BootWake
2. Dragging
3. DropLanding
4. PanelOpen
5. PanelClose
6. ErrorGlitch
7. UpdateInstalling
8. AchievementPop

每项写明：

1. 实现状态。
2. 触发方式。
3. 是否 experimental。
4. 是否有测试。

## 4. VFX Bus 验收

检查：

1. VFX 是否和状态机解耦。
2. 状态退出是否清理 VFX。
3. Sleep 是否关闭高频 VFX。
4. 粒子数量是否有限制。
5. 是否存在 DOM 无限累积风险。
6. 是否存在全屏 Canvas 重绘。

## 5. Transition 验收

检查：

1. 是否有差异化 transition duration。
2. Sleep 进入是否慢切。
3. Celebrate 进入是否快、退出是否松弛。
4. 高温 / 内存警报进入是否及时。
5. 状态切换是否会出现瞬移。

## 6. UI 联动验收

检查：

1. Quick Panel 是否从 pouch 弹性展开。
2. PanelClose 是否回收到 pouch。
3. MonitorBar 是否有胶囊流淌展开。
4. RollingValue 是否有数字 roll-up。
5. UI 动效是否和 CoreCat 动作冲突。

## 7. 性能验收

检查：

1. 常驻动画是否主要使用 transform / opacity。
2. 是否避免 top / left / width / height 高频动画。
3. VFX 是否可暂停。
4. LowPower 是否有效。
5. Performance Panel 是否可用。
6. 生产环境是否隐藏调试面板。

## 8. 资产验收

检查：

1. 是否有 asset manifest。
2. 是否有 fallback。
3. 是否有资源检查器。
4. 是否支持替换最终 PNG/SVG。
5. 是否仍保持分层，而非合成整图。

# 二、自动化测试与构建

请运行：

```powershell
corepack pnpm typecheck
corepack pnpm test:corecat
corepack pnpm build
```

如有失败：

1. 优先修复测试失败。
2. 不要跳过测试。
3. 如果是非本轮引入问题，也要明确说明。

# 三、代码风险扫描

请扫描并报告：

1. 是否存在未清理 timer / interval / RAF。
2. 是否存在状态机卡死风险。
3. 是否存在 VFX DOM 节点无限增长风险。
4. 是否存在生产环境 debug 面板泄露。
5. 是否存在资源路径硬编码风险。
6. 是否存在动画组件过度重渲染风险。

如发现小问题，请修复。  
如发现大问题，请先报告修改计划，再执行。
